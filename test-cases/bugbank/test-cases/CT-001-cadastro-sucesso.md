# CT-001 - Cadastro com sucesso

## Informações gerais

| Campo | Valor |
|-------|-------|
| **ID** | CT-001 |
| **Título** | Realizar cadastro com sucesso |
| **Status** | Pendente |

## Pré-condição

Página de cadastro do BugBank está carregada

## Dados de teste

| Campo | Valor |
|-------|-------|
| E-mail | teste@email.com |
| Nome | Jessica |
| Senha | Senha123 |
| Confirmação de senha | Senha123 |
| Conta com saldo | Sim |

## Passos

1. Clicar no botão "Registrar"
2. Preencher campo "E-mail" com `teste@email.com`
3. Preencher campo "Nome" com `Jessica`
4. Preencher campo "Senha" com `Senha123`
5. Preencher campo "Confirmação de senha" com `Senha123`
6. Selecionar opção "Criar conta com saldo"
7. Clicar no botão "Cadastrar"

## Resultado esperado

Página exibe mensagem de sucesso com o número da conta criada
