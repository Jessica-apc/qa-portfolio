# CT-003 - Criar conta sem saldo

## Informações gerais

| Campo | Valor |
|-------|-------|
| **ID** | CT-003 |
| **Título** | Criar conta sem saldo |
| **Status** | Pendente |

## Pré-condição

Todos os campos obrigatórios do formulário de cadastro estão preenchidos corretamente (consulte CT-001 para dados de exemplo)

## Passos

1. Verificar o switch "Criar conta com saldo?"
2. Não selecionar o switch (manter desativado)
3. Clicar no botão "Cadastrar"

## Resultado esperado

A conta é criada sem saldo disponível
