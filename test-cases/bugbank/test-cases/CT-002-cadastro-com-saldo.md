# CT-002 - Criar conta com saldo

## Informações gerais

| Campo | Valor |
|-------|-------|
| **ID** | CT-002 |
| **Título** | Criar conta com saldo |
| **Status** | Pendente |

## Pré-condição

Todos os campos do formulário de cadastro estão preenchidos corretamente (consulte CT-001 para dados de exemplo)

## Passos

1. Localizar o switch "Criar conta com saldo?"
2. Clicar no switch para ativá-lo (deve mudar para tom mais escuro)
3. Clicar no botão "Cadastrar"

## Resultado esperado

A conta é criada com saldo disponível
